# Hank and Yu Website

A Svelte-based personal website for the YouTube couple Hank and Yu.

## Tech Stack

- Svelte 4
- Vite
- Static site generation

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Fetch YouTube Videos

```bash
npm run fetch-videos
```

## Deploy

Build output is in `/dist` - deploy to any static host (Netlify, Vercel, GitHub Pages, etc.)

## Site Structure

- `/` - Home
- `/our-story` - Our Story page
- `/youtube` - YouTube page
- `/contact` - Contact page
- `/shop` - Links to shop.hankandyu.com
